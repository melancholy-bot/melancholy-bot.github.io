# Frenet Lab — 공간곡선과 TNB 틀 시각화 공간곡선 `r(t) = (x(t), y(t), z(t))` 위를 따라 움직이는 **접선(T)·주법선(N)·종법선(B)** 틀과 **곡률 κ**, **비틀림 τ**, **접촉원(osculating circle)** 을 실시간으로 보여주는 미분기하 실험실입니다. 다변수미적분에서 배우는 Frenet–Serret 틀을 "머릿속" 대신 눈으로 확인하려고 만들었습니다. --- ## 1. 폴더 구성 ``` frenet-lab/ ├── index.html # 화면 구조 ├── css/ │ └── style.css # 스타일(계측기 콘셉트) └── js/ ├── math-utils.js # 곡선 평가·수치미분·곡률/비틀림/TNB 계산 └── app.js # 3D 렌더링·자체 궤도 카메라·그래프·UI ``` 외부 라이브러리는 CDN으로 불러옵니다: 식 파서 **math.js**, 3D 렌더 **three.js**. --- ## 2. 실행 방법 (둘 중 아무거나) ### A. 가장 간단 — 파일 그대로 열기 `index.html` 을 더블클릭해서 브라우저로 엽니다. (인터넷 연결 필요 — CDN 라이브러리) ### B. 로컬 서버로 열기 (권장) 일부 브라우저는 보안 정책 때문에 로컬 서버에서 더 안정적으로 동작합니다. ```bash # 이 폴더 안에서 python3 -m http.server 8000 # 그 다음 브라우저에서 http://localhost:8000 접속 ``` --- ## 3. 웹으로 배포하기 (채점 7번: 직접 방문해 사용) 정적 사이트라 무료 호스팅에 그대로 올리면 됩니다. - **GitHub Pages**: 이 폴더를 저장소에 올리고 Settings → Pages → 브랜치 지정 - **Netlify / Vercel**: 폴더를 드래그&드롭 또는 저장소 연결 → 빌드 명령 없음(정적) 배포 후 주소를 보고서(PDF)에 적어두면 됩니다. --- ## 4. 사용법 - **곡선 선택**: 나선·세잎매듭·꼬인삼차곡선·비비아니·원뿔나선·토러스매듭 프리셋 클릭 - **직접 입력**: `x(t), y(t), z(t)` 에 식을 넣고 `t 범위` 지정 후 **그리기** (변수는 `t` 하나, `sin cos exp sqrt ^ pi` 등 사용 가능) - **재생/속도/위치 슬라이더**로 곡선 위 점을 이동 - **3D 화면**: 드래그로 회전, 휠로 확대/축소 - 우측 패널에서 **κ, 1/κ, τ, |r′|** 실시간 확인, 하단에서 **κ(t)·τ(t) 그래프** 확인 --- ## 5. 계산 원리 (요약) 수치미분(중심차분)으로 `r′, r″, r‴` 을 구한 뒤: - 곡률 `κ = |r′ × r″| / |r′|³` - 비틀림 `τ = (r′ × r″) · r‴ / |r′ × r″|²` - `T = r′/|r′|`, `B = (r′ × r″)/|r′ × r″|`, `N = B × T` - 접촉원: 반지름 `1/κ`, 중심 `r + (1/κ)N` 헬릭스 등 이론값이 알려진 곡선으로 검증했고, 곡률 오차는 약 10⁻⁹ 수준입니다.
